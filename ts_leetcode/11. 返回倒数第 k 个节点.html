# 题目描述

找出单向链表中倒数第 k 个节点。返回该节点的值。

# 示例

输入： 1->2->3->4->5 和 k = 2
输出： 4

# 数据范围

给定的 k 保证是有效的。

# 思路

关键点：双指针

快指针先走 k 步。随后快慢指针一起走。

时间复杂度：O(n)
空间复杂度：O(1)

# 代码

function kthToLast(head: ListNode | null, k: number): number {
  let fast = head
  for (let i = 0; i < k; i++) fast = fast.next
  let slow = head
  while (fast) {
    fast = fast.next
    slow = slow.next
  }
  return slow.val
}
